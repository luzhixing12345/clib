# libc

本仓库主要用于记录 C 标准库函数和 Linux 库函数的使用

同时本仓库也包含一个基础 C 工具库, 包括命令行参数处理和字符串处理. 笔者的另外两个项目 [coreutils](https://github.com/luzhixing12345/coreutils) 和 [binutils](https://github.com/luzhixing12345/binutils) 都使用到了本仓库的一些函数
